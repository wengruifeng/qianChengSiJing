# 微信隐私指引配置口径

本文件用于在微信公众平台填写小程序隐私指引时，对照本项目实际采集信息和使用目的进行配置。

## 建议声明的信息类型

### 1. 手机号码

- 使用场景：
  - 手机号登录
  - 客户身份识别
  - 价格查看申请审核
  - 订单联系
- 使用目的：
  - 完成登录认证
  - 完成客户审核
  - 完成订单履约和客服联系

### 2. 收货地址信息

- 包含内容：
  - 公司/店铺名称
  - 所在地区
  - 详细地址
  - 联系电话
- 使用场景：
  - 价格查看申请
  - 收货地址管理
  - 订单配送
- 使用目的：
  - 订单履约
  - 配送联系
  - 客户信息管理

### 3. 订单信息

- 包含内容：
  - 商品信息
  - 数量
  - 备注
  - 订单状态
  - 结算状态
- 使用场景：
  - 用户下单
  - 后台订单处理
  - 对账中心
- 使用目的：
  - 完成交易流程
  - 配送履约
  - 经营统计与对账

## 建议后台配置说明

- 本项目当前不接微信支付，一期不涉及支付账户信息采集。
- 本项目当前不申请通讯录、相册长期读写、定位等高敏权限。
- 图片上传由管理员在后台主动触发，属于经营管理功能，不面向普通用户常规开放。

## 提审时建议说明

可在审核补充说明中使用以下口径：

1. 本小程序为火锅串串食材订货与后台管理工具。
2. 用户侧仅采集实现登录、客户审核、地址管理和订单履约所必需的信息。
3. 后台管理能力仅对管理员/超级管理员开放。
