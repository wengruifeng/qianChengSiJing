# 微信审核说明建议

本文件用于准备微信官方审核时的补充说明。

## 小程序定位

前呈似景智链微信小程序，用于火锅串串食材客户浏览商品、申请查看价格、选购下单，以及商家后台进行商品、客户、订单、审核和对账管理。

## 建议服务类目口径

- 食品饮料
- 批发零售 / 订货采购相关类目

请以微信公众平台最终允许选择的实际类目为准。

## 审核路径建议

### 普通用户侧

1. 进入首页可浏览商品和分类
2. 未审核用户不可查看价格
3. 通过“我的 -> 价格查看申请”提交公司/店铺、地区和详细地址
4. 审核通过后可查看价格、加入选购、提交订单

### 管理后台

1. 管理员从“我的 -> 后台管理”进入后台
2. 可查看商品管理、客户管理、订单管理、审核中心、对账中心、店铺设置
3. 商品与店铺设置的修改需提交审核，由超级管理员审核通过后生效

## 审核测试说明建议

- 若审核员需验证后台路径，建议在审核备注中提供测试手机号和操作说明。
- 若审核版本不希望暴露联调入口，应确保“演示数据初始化”“清理演示图片路径”等调试工具在提审版本隐藏。

## 合规说明建议

1. 用户协议和隐私协议已在“我的 -> 协议及说明”提供。
2. 小程序当前不接微信支付，一期采用线下结算。
3. 用户手机号、地址和订单信息仅用于登录、审核、配送与订单履约。
